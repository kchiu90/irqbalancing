Welcome to irqservice Flask Homepage!

This service can provide following information:

  • irq_server
    • get_current_distribution
    • get_interrupt_by_cpu
    • set_affinity

Try following requests:

  • For current distribution:
    • http://localhost:5000/get_current_distribution?format=json
    • http://localhost:5000/get_current_distribution?format=raw
  • For CPU interrupts:
    • http://localhost:5000/get_interrupt_by_cpu (All CPU, 5 seconds)
    • http://localhost:5000/get_interrupt_by_cpu?cpu=1 (CPU1, 5 seconds)
    • http://localhost:5000/get_interrupt_by_cpu?cpu=2&second=3 (CPU2, 3seconds)
  • For smp_affinity modification:
    • http://localhost:5000/set_affinity/129?cpu=f (PUT request, server must run as root)

 

Enjoy!
